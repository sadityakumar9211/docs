## Current Status

**Core Engine** — Production-ready Go library with CEL-based condition evaluation and declarative effects.

**WASM Sandbox** — Full isolation for complex patch logic. Write policies in Rust, compile to WASM.

**Traffic Simulation** — Record request context and replay candidate policies before deploy.

**Operator UI** — Minimal web workbench for capture, simulation, scoped deploy, metrics, and rollback.
